### Scene 2: Example 1 — Standard Case

**Problem:** Solve $y' + 2y = e^{-x}$ with $y(0) = 3$.

1. **Step 1: Identify (Top Left):**
   - `Write` the problem: `y' + 2y = e^{-x}`.
   - Position at `LEFT * 4 + UP * 2.5`.
   - `Create` a `SurroundingRectangle` (Orange `#FB923C`) around `2` and label it `P(x)`.
   - `Create` a `SurroundingRectangle` (Sky Blue `#38BDF8`) around `e^{-x}` and label it `Q(x)`.

2. **Step 2: Integrating Factor (Center Left):**
   - `Write` calculation: `\mu(x) = e^{\int 2 dx} = e^{2x}`.
   - Position below Step 1. Color the final `e^{2x}` in `#38BDF8`.

3. **Step 3: Transformation (Bottom Left):**
   - `Write` the multiplication: `e^{2x}(y' + 2y) = e^{2x} \cdot e^{-x}`.
   - Use `TransformMatchingTex` to rewrite the left side as: `\frac{d}{dx}[e^{2x}y] = e^{x}`.

4. **Step 4: Integration & General Solution (Top Right):**
   - `Write` at `RIGHT * 3 + UP * 2`: `e^{2x}y = \int e^x dx`.
   - `Write` next line: `e^{2x}y = e^x + C`.
   - `Write` solution for $y$: `y(x) = \frac{e^x + C}{e^{2x}} = e^{-x} + Ce^{-2x}`.

5. **Step 5: Initial Value Problem (Center Right):**
   - `Write`: `y(0) = 3 \implies 3 = e^0 + Ce^0`.
   - `Write`: `3 = 1 + C \implies C = 2`.
   - `Write` final solution: `y(x) = e^{-x} + 2e^{-2x}`.
   - `Circumscribe` the final result in Mint Green `#4ADE80`.

6. **Step 6: Graph (Bottom Right):**
   - `Create` `Axes` from $x \in [0, 5]$ and $y \in [0, 5]$.
   - `Plot` the function $f(x) = e^{-x} + 2e^{-2x}$ in Mint Green `#4ADE80`.
   - `Create` a Dot at $(0, 3)$ to signify the initial condition.

### Scene 3: Example 2 — Non-Standard Form & Interval of Validity

**Problem:** Solve $x y' + 2y = 4x^2$ with $y(1) = 2$.

1. **Step 1: Normalization (Top Left):**
   - `Write` the equation: `x y' + 2y = 4x^2`.
   - `Create` a `CurvedArrow` pointing from a label `\div x` to the equation.
   - `Transform` the equation into: `y' + \frac{2}{x}y = 4x`.
   - Highlight `P(x) = \frac{2}{x}` in Orange `#FB923C`.

2. **Step 2: The Log Shortcut (Center Left):**
   - `Write`: `\mu(x) = e^{\int \frac{2}{x} dx} = e^{2\ln|x|}`.
   - `Write`: `\mu(x) = e^{\ln(x^2)}`.
   - `FadeIn` two Coral Red `#F87171` diagonal lines crossing out the `e` and `\ln`.
   - `Write` final factor: `\mu(x) = x^2`. Color this `#38BDF8`.

3. **Step 3: Solve (Bottom Left):**
   - `Write`: `\frac{d}{dx}[x^2 y] = x^2(4x) = 4x^3`.
   - `Write`: `x^2 y = \int 4x^3 dx = x^4 + C`.
   - `Write`: `y = x^2 + \frac{C}{x^2}`.

4. **Step 4: IVP & Interval (Right Side):**
   - Solve for C: `y(1) = 2 \implies 2 = 1^2 + \frac{C}{1^2} \implies C = 1`.
   - Final solution: `y = x^2 + \frac{1}{x^2}`.
   - `Create` a `NumberLine` at `RIGHT * 3 + DOWN * 1`.
   - Mark $x=0$ with a Coral Red `#F87171` cross `X`.
   - Mark $x_0 = 1$ with a Green dot.
   - `Highlight` the interval $(0, \infty)$ on the number line using a thick colored line segment.
   - `Write` text: `\text{Interval: } (0, \infty)` below the number line.
