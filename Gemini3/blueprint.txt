### **Scene 2: Step 1 - The Exactness Test**
**Layout Intent**: Title at top. A boxed rule on the left third. A specific example worked out on the right two-thirds.

| Visual Trigger | Narration |
| :--- | :--- |
| **Animation**: `Write` title: `\text{Step 1: The Exactness Test}`. Create a box on the **Left** containing `\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}`. | "First, we check if the equation is actually 'exact.' This happens if the partial of M with respect to y equals the partial of N with respect to x." |
| **Animation**: `Write` the example on the **Right**: `(2xy + y^2)dx + (x^2 + 2xy)dy = 0`. | "Let's try this with an example. Here's our equation. Can you spot our M and N functions?" |
| **Animation**: `Underline` `(2xy + y^2)` in **Sky Blue** and `(x^2 + 2xy)` in **Soft Yellow**. | "Exactly! M is our dx-term, and N is our dy-term." |
| **Animation**: `Write` `M_y = \frac{\partial}{\partial y}(2xy + y^2)` below the example. | "Now, let's cross-differentiate. We take the derivative of M with respect to y." |
| **Animation**: Dim the `x` in `2xy` and change its color to gray. `Transform` the expression into the result: `2x + 2y`. | "When we differentiate with respect to y, we treat x as a constant. So, 2xy becomes 2x, and y-squared becomes 2y." |
| **Animation**: `Write` `N_x = \frac{\partial}{\partial x}(x^2 + 2xy)` and show the result `2x + 2y`. | "We do the same for N, but with respect to x. Notice we get the exact same result: 2x plus 2y." |
| **Animation**: `Create` a large **Mint Green** (`#4ADE80`) checkmark between the two results. The `[Test]` circle in the map turns Green. | "Since they match, our equation is exact! We're cleared to move to the next step." |

| Visual Trigger | Narration |
| :--- | :--- |

### **Scene 4: Step 2 - Partial Integration**
**Layout Intent**: Checklist on the left third. Integration steps on the right.

| Visual Trigger | Narration |
| :--- | :--- |
| **Animation**: `Write` title: `\text{Step 2: Integrating for } \Psi`. Update Checklist: `1. Test [Green Check]`, `2. Integrate M [Active]`. | "Step 2 is to integrate M with respect to x to find our potential function, which we call Psi." |
| **Animation**: `Write` `\Psi(x,y) = \int (2xy + y^2) \, dx + g(y)` on the **Right**. | "We set up our integral using M. But here's the clever part: instead of adding a constant 'C', we add a function 'g' of 'y'." |
| **Animation**: `Transform` the integral into `x^2y + xy^2 + g(y)`. Highlight `+ g(y)` in **Purple**. | "Integrating with respect to x gives us x-squared-y plus x-y-squared. We include g-of-y because any function of y would have vanished during our x-derivative." |
| **Animation**: `Create` a `Brace` under `g(y)` with the text "Unknown function of y". | "Our goal now is to figure out what that unknown function g-of-y actually is." |

### **Scene 5: Step 3 & 4 - Matching and Solving g(y)**
**Layout Intent**: Split screen. Left side shows current Psi. Right side shows the target N.

| Visual Trigger | Narration |
| :--- | :--- |
| **Animation**: `Write` title: `\text{Step 3 \& 4: Finding the Unknown}`. **Left Side**: `\Psi = x^2y + xy^2 + g(y)`. **Right Side**: `\text{Target: } \Psi_y = N`. | "To find g-of-y, we'll differentiate our current result with respect to y and compare it to our original N function." |
| **Animation**: `Apply` `\frac{\partial}{\partial y}` to the **Left Side**. Show `x^2y \rightarrow x^2`, `xy^2 \rightarrow 2xy`, and `g(y) \rightarrow g'(y)`. | "Differentiating with respect to y, x-squared-y becomes x-squared, x-y-squared becomes 2xy, and g-of-y becomes g-prime." |
| **Animation**: `Write` the equality in the **Center**: `x^2 + 2xy + g'(y) = x^2 + 2xy`. | "Now, we set this equal to our original N. Take a look at both sides—do you see anything that might cancel out?" |
| **Animation**: **Pause** (2 seconds). | (Rhetorical Pause for learner to identify terms). |
| **Animation**: `Create` **Coral Red** `Cross` lines striking out `x^2` and `2xy` on both sides. | "That's right! The x-squared and 2xy terms appear on both sides, so they cancel out completely." |
| **Animation**: `Write` `g'(y) = 0 \Rightarrow g(y) = C_1` below the cancellation. | "This leaves us with g-prime equals zero. Integrating that gives us a simple constant, which we'll call C-1." |
| **Animation**: Update Milestone Map: `[Integrate]`, `[Differentiate]`, and `[Solve g(y)]` all turn **Mint Green**. | "We've solved the mystery of the unknown function! Now, let's put it all together." |
